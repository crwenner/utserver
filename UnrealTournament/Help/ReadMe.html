Nomad Rocket Bots  

New Nomad Rocket Bot Mutator Released!
Posted by Britton at 8:20 | Discuss In Unreal 2 CTF Forum


Download the Nomad Umod Here.

The Nomad bot from Star Trek makes a surprise appearance in Unreal Tournament.  He has his standard "I am Nomad" introduction, and then he begins killing people.  The alt fire on the rocket launcher will produce this wonderful guest.  He lasts about 30 seconds.  The Nomad requires only one rocket ammo for creation and then he does the shooting for you, using your ammo as his supply.  He has 90 health and explodes when he dies.  Having several nomads close together cause a violent chain reaction.  What more could you ask for?  

How about seeking missiles that hunt down the enemy?  Oh yes he has those too.  Monster kills have never been easier.

Installation Notes

Umod Installation
----------------------
Unzip the files in this archive to a temporary folder and then double click the Nomad.umod file. This will open the umod wizard.  From the wizard select the UnrealTournamnet directory and click install.

Start a new multiplayer game. On the match tab is a "mutators" button. clicking the mutator button brings up the configure mutator window. The Nomad Rocket Bots mutator should be displayed on the left. Double clicking the text value will add it to the right. Close the window and Start the game. You are ready to go!


Manual Installation
------------------------
If you are installing manually then copy the following files to the unrealTournamnet directory:
c:\UnrealTournament\System\NBot.u
c:\UnrealTournament\System\NBot.int

Then follow the instructions below to add the SmartRockets server package entry in the unreal tournamnet ini.

The unrealtournament.ini file ( usually located in c:\UnrealTournament\system ) needs the following entry was added in the [Engine.GameEngine] section. First find the 
[Engine.GameEngine] section, then look for the "NBot" entry at the bottom. If it is not there add it as seen below:


[Engine.GameEngine]
ServerPackages=NBot

Start a new multiplayer game. On the match tab is a mutator button. clicking the mutator button brings up the configure mutator window. The Nomad Rocket Bots mutator should be displayed on the left. Double clicking the text value will add it to the right. Close the window and Start the game. You are ready to go!
